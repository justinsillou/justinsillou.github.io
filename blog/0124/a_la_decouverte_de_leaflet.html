A la découverte de Leaflet

Leaflet est une bibliothèque JavaScript légère et simple à utiliser pour créer des cartes interactives dans vos applications web. Elle offre une interface conviviale et des fonctionnalités puissantes pour l'affichage de cartes, le marquage de points, la gestion des événements, et bien plus encore.

Installation de Leaflet

Pour commencer, ajoutez la bibliothèque Leaflet à votre projet en incluant le fichier CSS et JavaScript :

        
            <link rel="stylesheet" href="https://unpkg.com/leaflet/dist/leaflet.css" />
            <script src="https://unpkg.com/leaflet/dist/leaflet.js"></script>

Création d'une carte simple

Utilisez le code suivant pour créer une carte Leaflet de base :

        
            // Initialisation de la carte
            var map = L.map('map').setView([48.8566, 2.3522], 13);

            // Ajout d'une couche de tuiles OpenStreetMap
            L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
                attribution: '© OpenStreetMap contributors'
            }).addTo(map);

Marquer un emplacement sur la carte

Vous pouvez ajouter des marqueurs à la carte pour indiquer des emplacements spécifiques :

        
            // Ajout d'un marqueur à un emplacement spécifique
            var marker = L.marker([48.8566, 2.3522]).addTo(map);
            marker.bindPopup('Paris, France').openPopup();

Ceci ajoutera un marqueur et une infobulle
